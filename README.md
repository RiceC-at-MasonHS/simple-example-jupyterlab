# Simple Example Jupyter Notebook
This notebook is just meant for a bare-bones example with Jupyter Notebook. 

For anyone on a Chromebook (most HS students) You can use [Jupyter's Online Lab](https://jupyter.org/try-jupyter/lab/) to play with this example.
Download the [simple-example.ipynb](simple-example.ipynb) file, and you can upload it to the Jupyter Online Lab.
